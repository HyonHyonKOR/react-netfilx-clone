## 環境機能一覧

検索機能：入力したキーワードと連関した作品を 20 個検索　<br>
　非同期処理；人気作品 10 個の画像を 15 秒ごとに表示　<br>
　 UI：モーダルウィンドウ/カルーセル/検索アイコンなど　<br>
